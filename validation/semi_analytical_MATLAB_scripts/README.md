# Semi-Analytical Solution for GPR: The MATLAB Scripts

This folder contains MATLAB scripts for generating semi-analytical solutions based on electromagnetic wave propagation in theoretical simplified geophysical models.

## Authors

Many of these scripts were originally written by [Prof.dr.ir. E.C. Slob](https://www.tudelft.nl/en/staff/e.c.slob/)([Google Scholar](https://scholar.google.com/citations?user=ElIrhdkAAAAJ&hl=en)), of the Delft University of Technology. To these scripts, Chaitanya Singh added minor functionality to output CSVs in a relevant format. He also worked on extending the solution of the half-space model to add the scattered field from a sphere anomaly.

## Usage

As the data is already computed and provided in the `data` folder of this repository, you can start testing functionalities of `elfe3D_GPR` without having a MATLAB license. However, if you would like to change model parameters, you can easily do so by opening the `.m` files in MATLAB and adjusting them as needed.
