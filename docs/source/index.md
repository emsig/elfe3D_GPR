# elfe3D_GPR Documentation

Welcome to the official documentation for `elfe3D_GPR`, the 3D finite-element forward modelling framework for ground penetrating radar (GPR).

This site covers the overall workflow, the Python I/O layer, the solver theory, and the example notebooks that demonstrate the core capabilities of the code.

The documentation is organized so that users can quickly get started with notebooks and Python, while researchers can follow the scientific and implementation details of the solver.
