Fortran API Reference
=====================

The Fortran source tree in `elfe3D_GPR/` contains the core solver implementation.
The generated API documentation is published as integrated Doxygen HTML under the same site.

<a href="doxygen/index.html">Open the Fortran API documentation</a>

What is generated
-----------------

The integrated documentation includes:

- solver entry points
- input file semantics
- module descriptions
- core numerical routines
- generated source browsing and module dependency graphs

Core modules
------------

Important modules include:

- `elfe3d_gpr` (solver executable driver)
- `mod_define_model.f90`
- `mod_solvers.f90`
- `mod_calculate_matrices.f90`
- `mod_read_mesh.f90`
- `mod_error_estimates.f90`

Notes

This page is the RTD landing page for the Fortran API docs. The actual API content is generated by Doxygen and exposed under `doxygen/index.html` so it is visible within the same website.
